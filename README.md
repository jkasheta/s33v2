## Features

- Animated terminal boot sequence with typing effects
- Interactive menu system (click or keyboard navigation 1-4)
- Password-protected Intel section (access code: S33ACCESS)
- Mobile-first design (375px viewport)
- Zero dependencies, pure HTML/CSS/JavaScript

## Menu Options

1. **Mission Brief** - Studio background and recent work
2. **Services** - Production capabilities
3. **Intel** - Internal notes (password protected)
4. **Contact** - Get in touch

## Technical Stack

- Static HTML/CSS/JavaScript
- Deployed via GitHub Pages
- No build process required

## Local Development

Simply open `index.html` in a web browser. No build step or dependencies needed.

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.
